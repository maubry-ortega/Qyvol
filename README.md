## ✨ Primeros Pasos

### Instalar CLI

```sh
cd cli
cargo install --path . --bin qyv
```

### Ejecutar un ejemplo (Rust)

```sh
qyv run examples/hello/hello.qyv
```

### Ejecutar un ejemplo (Go)

```sh
qyv run ejemplos_funcionales/ejemplo/ejemplo.qyv
```
